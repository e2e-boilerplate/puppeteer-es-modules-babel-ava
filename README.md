# Puppeteer Boilerplate
    
Puppeteer end-to-end test automation boilerplate, Using [ES Modules](https://hacks.mozilla.org/2018/03/es-modules-a-cartoon-deep-dive/), [Babel](https://babeljs.io) and [AVA](https://github.com/avajs/ava). Clone or fork this repository.
    
## Getting Started
    	1. git clone git@github.com:e2e-boilerplate/puppeteer-es-modules-babel-ava.git
    	2. cd puppeteer-es-modules-babel-ava
    	3. npm install
    	4. npm run test
        
    
For more boilerplate click [here](https://github.com/e2e-boilerplate/utils/blob/master/docs/implemented.md)
